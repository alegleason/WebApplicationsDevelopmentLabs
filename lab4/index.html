Preguntas

  • ¿Qué diferencias y semejanzas hay entre Java y JavaScript?
    Algunas diferencias son los tipos de comandos en ambos lenguajes, que difieren bastante, por ejemplo el Scanner en java y el prompt en JavaScript. JS es un lenguaje interpretado y Java compilado. Una similitud podría ser que Java es un lenguaje orientado a objetos, y JavaScript debe simular muchas de las características de orientación a objetos por medio de prototipos. [1]

Ejercicios


*El ejercicio 6 brinda solución a un problema que me enfrento diariamente. En mi página de ventas realizo continuamente rifas de perfumes de manera gratuita entre los seguidores que crecen diariamente en mi cuenta. En internet hay muchas ruletas pero yo quiero asegurarme que sea realmente al azar, por lo que lo mejor sería programarla yo mismo. La solución es crear dicho mecanismo que nos dé ganadores al azar, y que nos diga qué premio se lleva, previemante se hace una votación de qué perfumes quieren que se rifen para ser ingresados, no hay problema en que los ganadores se lleven el mismo o diferente premio, ya que es al azar la elección del premio también. El número de participantes puede variar, pero siempre hay tres ganadores.
