Preguntas

¿Qué importancia tiene AJAX en el desarrollo de RIA's (Rich Internet Applications?

Que permite que las aplicaciones desarrolladas de Internet funcionen tan bien como las aplicaciones de escritorio

¿Qué implicaciones de seguridad tiene AJAX? ¿Dónde se deben hacer las validaciones de seguridad, del lado del cliente o del lado del servidor?

Se tienen que seguir haciendo las validaciones de seguridad del lado del servidor, asegurarnos de prevenir ataques de Cross Side Scripting, SQL Inyection y demás.

¿Qué es JSON?

JavaScript Object Notation: Es un formato de texto plano, un conjunto de atributos con valores. Que en lugar de regresar texto puede regresar atributos que son interpretados más fácilmente por JavaScript y colocados en el HTML.
